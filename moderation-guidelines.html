Moderation Guidelines

Last Updated Monday, June 21st.

What Constitutes a Mute

Muting is when a user loses access to every channel in Plover except #ban-lands. Mutes are also referred to as shadow-bans. Muting is the lowest punishment and can be assigned by anyone with Moderator perms.

Mutes can be given when:

  1. A user messed up counting in #counting on purpose. (3h)
  2. A user needs a timeout during a conversation in a general chat (2d)
    • A timeout is needed when a user is using slurs to make people mad, getting heated over politics and, treating people or whole groups. Look for overall aggravation.
  3. A user is DM Advertising (10d)
  4. A user is leaking Plover assets and we need to investigate (3y)
  5. A user is spam pinging (1d)
  6. A user is breaking rules but does not qualify for a ban or kick (30d)
  7. A user is talking about leaked Plover tech they have or asking people for free tech (3d)
  8. A user is using filter bypass to say slurs (N**ger, Tra*ny, and Fag*ot.) (2d)
  9. A user is talking badly about another person (5h)
  10. A user is misusing Plover Marketplace (1d)

The (#d) after is the max amount of time the punishment can last!
Example: 1d = 1 day. 5h = 5 hours.
When a user is muted they should also be warned!

What Constitutes a Warning

Warning a user adds a warning to their account. Once a user had 6 warning they will be temp banned from Plover for 3 days. Warns should be used as a way to let people know they are doing something wrong and should stop.

Warns can be given when:

  1. A user has messed up the count in #counting on purpose twice.
  2. When a user is muted they should also be warned.
  3. When a user spams messages to win a giveaway.
  4. Inappropriate emoji reactions such as NSFW emojis showing gore and sexual content in the News and Miscellaneous category.
  5. A user is found spamming / saying inappropriate things such as slurs in the Plover group wall.
  6. A user is spam creating customer support tickets (up to 3 times) If the user continues after a warning mute them for 1 day.
  7. A user is misusing the Plover website to submit fake / spam orders.

What Constitutes a Kick

Kicking a user removed them from the Plover Discord. The user is free to rejoin whenever they want.

Kicks can be given when:

  1. A user has 3 warnings but continues to misbehave.
  2. A user continues to DM advertise after they have been muted for it.
  3. A user continues to spam ping after they have been muted for it.

What Constitutes a Temporary Ban

Temporary Banning is when a user is banned for a short amount of time. When that time is up they are allowed to rejoin the Plover Discord.

Temporary Bans can be given when:

  1. A user has been warned 6 times (Auto Mod will cover this).
  2. A user has been kicked but rejoined and continued their behavior. (1w)

The (#w) after is the max amount of time the punishment can last.

What Constitutes a Ban

Banning is when someone is no longer welcome in Plover. The user will no longer be able to participate in Plover events in any Plover related games either. Usually someone who is banned is also Blacklisted.

Bans can be given when:

  1. A user has been temp banned and continues their behavior.
  2. A user goes against the Buyer Terms of Service.
  3. A user is promoting speech that expresses hate or encourages violence towards a person or group based on something such as race, religion, sex, or sexual orientation.
  4. A user is leaking Plover tech.
  5. A user is under the age of 13.
  6. A user violates the Discord or Roblox terms of service.
  7. A user is spreading child pornography or any NSFW content.
  8. A user is spreading leaked assets from any group in general chats.
  9. A user is doxxing someone else with malicious intent.
  10. A user is found attempting to break into staff member emails.
  11. A user is attempting a Denial of Service attack, or any malicious attack that may break the Computer Abuse and Fraud Act on any Plover assets, domains, etc.
  12. A user modifies software / mesh identification to create a new untraceable / cracked product.
  13. A user uses exploits in Plover SMP or raids spawn areas.
  14. A user injects exploits in Plover owned games.
  15. A user or group attempt to overthrow Plover or any subsidiaries it owns.
  16. A user is an alternative account.
  17. A user is harassing others in direct messages. We define harassment as persistent attacks and criticism causing worry and distress.

Plover Executives may request someone is banned at any time for any reason. An anonymous vote must be held by all Executive Management+ to approve it.

Who does this apply to?

This document applies to all users in the Plover Discord and in the group. If you feel moderation action taken against you is unfair, you may appeal. Appeals are not applicable if you have leaked Plover assets, are under the age of 13, distributed child pornography, or you are banned via vote by a Plover Executive.
